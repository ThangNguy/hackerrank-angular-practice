💡 Cách thêm bài tập mới

  1. Tạo component mới trong src/app/components/exercises/
  2. Code UI và logic cho bài tập của bạn
  3. Thêm route trong app.routes.ts
  4. Đăng ký bài tập trong ExerciseService (initializeDefaultExercises)
  5. Component sẽ tự động xuất hiện trong Dashboard!

💡 Tip: Tham khảo component unit-converter để xem pattern!
